### Week 7 Objectives:

* Understand the fundamentals of Infrastructure as Code (IaC).
* Learn AWS CloudFormation concepts: templates, stacks, resources.
* Explore YAML/JSON CloudFormation template structure.
* Practice deploying infrastructure using CloudFormation.
* Learn how to update, delete, and troubleshoot CloudFormation stacks.
